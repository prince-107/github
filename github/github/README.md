# Project Documentation

## Overview
This project is a simple website for Sharnbasva University, designed to provide information and contact options for users. It consists of a main page and a contact page, both styled with CSS.

## Files

### 1. `index.html`
- **Purpose**: Serves as the main page of the website.
- **Features**:
  - Welcome message
  - Navigation links to other pages (Home, About, Contact)
  - An image
  - A descriptive paragraph
- **Styling**: Includes CSS for basic styling of headings and paragraphs.

### 2. `contact.html`
- **Purpose**: Provides a contact page for users to reach out.
- **Features**: Will include a contact form and relevant information.
- **Styling**: Will have CSS styling similar to `index.html`.

## Usage
To view the website, open `index.html` in a web browser. Use the navigation links to access different pages. The contact page will allow users to get in touch with the university.
